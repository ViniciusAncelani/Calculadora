# Calculadora

### 🔢 Calculadora Simples
Este projeto é uma calculadora básica desenvolvida com HTML, CSS e JavaScript, ideal para treinar lógica de programação e manipulação do DOM.

✨ Funcionalidades
Adição, subtração, multiplicação e divisão

Interface responsiva e amigável

Limpar operação atual (C)

Exibição em tempo real dos números digitados

### 🚀 Tecnologias Utilizadas
HTML5

CSS3

JavaScript (puro)

### 🎯 Objetivo
Este projeto tem como objetivo reforçar os fundamentos de JavaScript, principalmente:

Eventos de clique (addEventListener)

Manipulação de strings e arrays

Atualização dinâmica do conteúdo com innerText

Uso de funções e estruturas condicionais
